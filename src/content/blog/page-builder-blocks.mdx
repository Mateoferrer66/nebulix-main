## Main content

Each page features a primary MDX content field, serving as the core content area. We've integrated the @remark-embedder/transformer-oembed plugin for seamless embed support. Adding images is also effortlessly accomplished within this content space.

![Cosmic MDX editor](/src/assets/embeds.png)

## What are blocks?

In Nebulix, blocks are fundamental building elements that enable yu to construct custom web pages. These blocks serve as the foundational components for your webpage's layout and content.

What makes Nebula particularly powerful is that each block can be individually styled using custom classes, allowing you to fine-tune their appearance, adjust container widths, and even modify their surfaces.

### View Transitions

This theme harnesses the power of the View Transitions API. Please note that transition names for elements must be unique. Therefore, only one page transition can be active on the same page. To enable animations, you'll find an option on every block that can link to another page or image.
